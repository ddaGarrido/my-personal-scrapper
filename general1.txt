API de Acesso e coleta de dados em sites de utilidades.
Nome: my-personal-scrapper
Objetivo:
A API de Acesso e Coleta de dados em sites fornece funcionalidades essenciais para realizar o login em um determinado site, coletar informações de boletos,
salvar essas informações e listar os dados coletados em uma biblioteca.
